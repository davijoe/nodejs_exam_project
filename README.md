# Gummiand

Gummiand is an exam project for the course "Node.js Full Stack Development".

It is a simple Kanban app with a real time chat as well.

## Stack
- Node.js
- MongoDB
- Socket.io


## Features
- Sign up
- Login / logout
- Account deletion. 
- Includes owned tasks
- Manage tasks (CRUD)

- sorting & filtering
- Query Tasks on:
    - completed
    - creation date
- limit (custom)
- pagination (skip)
- Upload files
